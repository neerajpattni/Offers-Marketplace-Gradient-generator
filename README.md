# Offers Gradient Generator

A standalone ClearScore Offers marketplace gradient generator.

## What it does

- Generates Contentful-ready gradient arrays.
- Provides existing Offers marketplace presets for context.
- Lets users explore new hue families while keeping the final two stops locked to `#364665`.
- Previews gradients in a portrait viewport with optional marketplace cards.

## Run locally

Open `index.html` in a browser, or serve the folder with any static file server.

```sh
python3 -m http.server 8765
```

Then visit:

```text
http://127.0.0.1:8765
```

## Deploy

This project has no build step. It can be hosted directly with GitHub Pages or any static hosting provider.
